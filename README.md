## Overview
AWS AI Learning Planner is a generative AI-based application built with AWS PartyRock. It creates personalized learning roadmaps, structured revision plans, and study recommendations based on user input such as skill level and available study time.

The project demonstrates practical use of prompt engineering and foundation models to build real-world AI applications.
