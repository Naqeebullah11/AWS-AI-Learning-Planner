# AWS AI Learning Planner

## Live Application
https://partyrock.aws/u/naqeeb19/jPWw3F43_/Smart-AI-Learning-and-Career-Planner

## Overview

AWS AI Learning Planner is a generative AI-based application built using AWS PartyRock. It generates personalized learning roadmaps, structured revision plans, and study recommendations based on user inputs such as skill level, goals, and available study time.

## Key Highlights

* Personalized AI-generated learning roadmaps
* Structured revision and study planning
* Adaptive recommendations based on user input
* Simple and interactive AI-powered interface

## Tech Stack

* AWS PartyRock
* Generative AI (Foundation Models)
* Prompt Engineering

## Project Purpose

This project demonstrates practical application of prompt engineering and foundation models to solve real-world learning and productivity challenges using generative AI.

## Note

This project was built as part of the AWS AI & ML Scholars – AWS AI Practitioner Challenge (Udacity) where I explored generative AI, prompt engineering, and AWS PartyRock.
